Dreambox

The JS image box of my dream.

View project onGitHub

Hi-res images viewer

Unlike the the others lightboxes dreambox does not enlarge a continer of the images to some another fixed size but let the image to be moved and zoomed to any desired scale and position

Besides zooming and moving the plugin supports fullscreen mode of the modern browsers

Disables right clicking and saving the image. Eliminates IMG tag from the HTML code

Use

As usual you need to include the plugin into your header 


<script src="https://rawgithub.com/dsame/dreambox/master/src/js/dreambox.js"></script>

and apply the paramaters to some HTML blocks containing the references to the images: 


<script>
jQuery(function(){
$('.dreambox').dreambox({
// Alternative to background color CSS property of the container
	bgColor:'#bb0',
// Will be IMG tag used or background-image CSS property to render the image
	useBackground:false,
// "cp" stands for "Control Panel"
	cpShowDuration:300,
	cpHideDuration:2000,
	cpHideTimeout:2000,
	cpPanelOpacity:0.6,
	cpButtonsOpacity:0.6,
// Directs the plugin to do not load Raphael libray. May be useful if that script already loaded manually 
	dontLoadRaphael: false,
// Delta of the zooming in the percent of the current size
	zoomingSpeed:10
})
})
</script>

and put the container with the image somewhere in your code:


<div class="dreambox" style="width:690px;height:400px"><img src="images/3683351134_b58357fac1_o.jpg"></img></div>
